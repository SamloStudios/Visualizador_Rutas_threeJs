-- PARA EXECUTAR EL CODIGO POR PRIMERA VEZ:
1. Es necesario tener instalado node.js y npm - https://nodejs.org/es

2. Comando para instalar los modulos:
    npm install
Se debe executar dentro del directorio raíz del proyecto (donde está package.json)

3. Despues de instalar los modulos correspondientes, se compila la pagina con:
    npm run dev
